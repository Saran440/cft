Reminder Configuration

You can make generalised reminder with the configuration of the reminders as per the screen below. Configuration contains model and field configuration for the reminder. Reminders will be based on following configuration. Days : Next upcoming days configured Today : Event on the current date Current Month : Event in the current month Next Month : Event in the next month Colour : It will be the background colour on the widget.
